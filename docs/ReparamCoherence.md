# 定理 C(再パラメータ化コヒーレンス)— 完全証明

作成: 2026-07-16。新定理プログラムの**確定第 1 号**。
機械的証拠: LibStrictness.lean §1(正例)・§8(完全性の負例 +
De Morgan 双対)。

## 1. 主張

**定理 C.** 本カーネルのアルゴリズム的等式(NbE 誘導の変換)において、
generic な文脈 Γ = (A : U, a b : A, p : Path A a b) と、
n 個の区間変数上の自由 De Morgan 代数 DM(i₁,…,iₙ) の元 φ, ψ で
角の値が一致するもの(全ての ε ∈ {0,1}ⁿ で φ(ε) = ψ(ε))について:

  ⟨i₁…iₙ⟩ p(φ) ≐ ⟨i₁…iₙ⟩ p(ψ)  ⟺  φ = ψ in DM(i₁,…,iₙ)

**コヒーレンス読み**: generic な道への接続代数の作用は自由 De Morgan
代数に**忠実**である。すなわち再パラメータ化の間の定義的コヒーレンスは
De Morgan 等式そのものであり、それ以上でもそれ以下でもない。
(接続層の完全な特徴付け — 「どの図式が可換か」への層別回答の第 1 層。)

## 2. カーネル節の典拠

証明は以下の実装節のみに依存する(Semantics.lean / Interval.lean):

- (E) `eval` の papp 節: `p @ r` は p が中立なら
  `vne (papp p a b r̂)`(r̂ = r の IVal 値)。ただし r̂ が
  恒等的 0/1 なら端点 a/b に**崩壊**(vpapp の isZero/isOne 分岐)。
- (C1) `convIBinder`: 2 つの区間束縛は generic レベル l で実体化して
  比較される。
- (C2) `convNe` の papp 節: `papp p _ _ r ≐ papp p' _ _ r'` ⟺
  `convNe p p' ∧ r.equiv r'`(端点注釈は比較されない)。
- (D) `IVal.equiv r s = (dnf r == dnf s)`、ここで dnf は
  反鎖標準形(自由分配束 + 対合の標準形)。**dnf の健全・完全性**
  (r = s in 自由 DM 代数 ⟺ dnf r = dnf s)は自由 De Morgan 代数の
  標準形定理(Interval.lean の設計典拠)。

## 3. 証明

n 変数の場合を示す(機械的証拠は n = 1, 2)。両辺は n 重 plam。

**(⇐ 健全性)** φ = ψ in DM とする。conv は (C1) により両辺を
generic レベル l₁,…,lₙ で実体化する。代入 iₖ ↦ var lₖ は
DM 代数準同型なので φ(l⃗) = ψ(l⃗) in DM(l₁,…,lₙ)、
よって (D) により dnf が一致。
- φ(l⃗) が恒等 0/1 の場合: ψ(l⃗) も同じ定数、両辺とも (E) により
  同じ端点値に崩壊 ✓。
- そうでない場合: 両辺とも中立 papp、(C2) の r.equiv が (D) で真 ✓。

**(⇒ 完全性)** conv が受理したとする。generic 文脈では p, a, b は
相異なる変数(中立)である。
- 両辺とも非崩壊(φ(l⃗), ψ(l⃗) が非定数)の場合: (C2) により
  `φ(l⃗).equiv ψ(l⃗)` が真、(D) により dnf 一致、標準形定理により
  φ(l⃗) = ψ(l⃗) in DM(l⃗)。代入 iₖ ↦ lₖ は自由代数間の同型
  (生成元を生成元へ)なので φ = ψ ✓。
- 片方のみ崩壊(例: φ(l⃗) 恒等 0、ψ(l⃗) 非定数)の場合:
  左辺は変数 a、右辺は中立 papp — conv の構造比較は
  var と papp を区別するので**受理されない**(前提に矛盾)。
  よってこのケースは生じない ✓。
  (機械的確証: `i∧¬i` vs `0`、`i∨¬i` vs `1` の負 probe —
  自由 DM 代数に排中律がないことがそのまま定義的等式に反映される。)
- 両方崩壊の場合: 両辺とも端点変数。受理 ⟺ 同じ変数 ⟺
  φ(l⃗), ψ(l⃗) が同じ恒等定数。角の値一致の仮定と合わせ、
  定数 DM 元として φ = ψ ✓。∎

## 4. 系と注意

- **系 1(1 変数の分類)**: generic ループ p : Path A a a 上、
  ⟨i⟩p(φ(i)) の定義的等価類は自由 DM 代数 DM(i) の
  端点条件を満たす元と 1:1(6 元中、端点 (0,1) は {i}、(0,0) は
  {0, i∧¬i} — 後者 2 つは**区別される**)。
- **系 2**: symm(= ¬ 前合成)・接続(∧/∨ 前合成)のみから成る
  全ての図式のコヒーレンスは決定可能かつ dnf で判定される。
- **注意(範囲の限界)**: 本定理は再パラメータ化層のみを扱う。
  trans/hcomp を含む層は本質的に弱い(カタログの負例:
  trans refl refl ≢ refl 等)— そこへの拡張がターゲット 2 の
  次段階(hcomp-セルを法とする正規形理論)。
- **注意(理論一般化)**: 主張と証明は「papp が端点崩壊し、
  中立 papp の比較が区間の意味的等価で行われる」任意の
  CCHM/ABCFHL 型実装・理論に持ち上がる。理論版では (C2)(D) を
  当該理論の変換規則+正規化定理に置換する。

- 新規性: 「接続層の定義的コヒーレンスの完全な特徴付け」としての
  明示的な言明・証明は管見の限り文献にない(要最終確認)。
  ターゲット 2(接続代数の Mac Lane 型コヒーレンス)の基底層を
  確定させる基礎石。
- 機械的証拠: 正例 5 種(冪等・対合・吸収・零・De Morgan 双対)+
  負例 2 種(排中律の不在)が LibStrictness で検証済み。
